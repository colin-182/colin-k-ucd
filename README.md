# Music World – Website Project

This project is a 4–page website focused on music news, concerts, and live event highlights in Ireland.  
It was created as part of a web design module to demonstrate skills in HTML, CSS, responsive design, layout planning, and multimedia integration.

---



---

## Features Implemented (as required by the assignment)

- Appropriate **meta tags** for accessibility and SEO awareness
- **Consistent header**, logo area, and navigation across all pages
- **External CSS stylesheet** (`style.css`) for presentation and layout
- **Responsive page layout** suitable for mobile and desktop devices
- **Top 10 music chart** using an HTML table (`index.html`)
- **Concert information table** with horizontal scroll on smaller screens (`concerts.html`)
- **Clickable email hyperlink** (located in footer or news content)
- **External link** to Ticketmaster for ticket purchasing
- **Image gallery** bonus feature implemented (`gallery.html`)
- Tested and displays correctly on **Chrome, Edge, Firefox**, and mobile browsers
